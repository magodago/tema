# Diseño del proyecto

## Objetivos
- Comprender los elementos fundamentales del diseño de proyectos.
- Identificar y analizar las necesidades y objetivos del proyecto.

## Contenido
El diseño del proyecto es un paso crítico que establece el marco para su implementación exitosa. En esta lección, se enfatiza la importancia de definir claramente los objetivos del proyecto y entender las necesidades del cliente o usuario final. Se abordarán técnicas para recopilar información relevante sobre el contexto del proyecto, incluyendo investigación de mercado, análisis de requisitos y definición de scope. Estos pasos permiten crear un plan detallado que guiará todo el proceso de desarrollo.

## Ejercicio
El estudiante deberá seleccionar un proyecto real o hipotético y realizar una recopilación de información basada en las técnicas aprendidas, incluyendo la identificación de los objetivos del proyecto, la definición del scope y la descripción de las necesidades del cliente. Este ejercicio permitirá a los estudiantes aplicar teoría a práctica y prepararse para etapas posteriores del diseño.

## Resumen
- Definir claramente los objetivos del proyecto.
- Identificar y analizar las necesidades del cliente o usuario final.
