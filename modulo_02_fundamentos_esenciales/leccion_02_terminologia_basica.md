# Terminologia basica

## Objetivos
- Comprender y utilizar correctamente la terminología básica del curso.
- Identificar los conceptos fundamentales que se tratarán a lo largo de las lecciones.

## Contenido
La terminología es un elemento crucial en cualquier campo de estudio, ya que proporciona una base sólida para el entendimiento. En este módulo, se introducirá la vocabularia fundamental que será utilizada constantemente a lo largo del curso. Se tratarán conceptos como "variable", "constante" y "operador". Estos términos son esenciales para comprender cómo funciona el lenguaje de programación y realizar cálculos básicos.

Además, se explicarán las diferencias entre "lógica booleana" y "lógica proposicional", conceptos que forman la base del pensamiento computacional. La comprensión de estas definiciones permitirá a los estudiantes abordar problemas complejos con mayor facilidad y precisión.

## Ejercicio
Escribe una breve descripción de cada uno de los siguientes términos: variable, constante, operador, lógica booleana y lógica proposicional. Luego, proporciona un ejemplo práctico para cada término.

## Resumen
- La terminología básica es fundamental para el éxito en el curso.
- Los conceptos clave incluyen "variable", "constante" y "operador".
- Se deben comprender las diferencias entre "lógica booleana" y "lógica proposicional".
