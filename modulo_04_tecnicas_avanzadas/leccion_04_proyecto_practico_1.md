# Proyecto practico 1

## Objetivos
- Desarrollar habilidades prácticas en la implementación de algoritmos de aprendizaje automático.
- Familiarizar a los estudiantes con el uso de bibliotecas de machine learning en Python.

## Contenido
En esta lección, se profundizará en las técnicas avanzadas del aprendizaje automático, enfocándose en su aplicación práctica. Los estudiantes aprenderán cómo implementar modelos de regresión y clasificación utilizando la biblioteca scikit-learn en Python. Se abordarán conceptos como el ajuste de hiperparámetros, validación cruzada y selección de características para mejorar la precisión del modelo.

Se discutirá también la importancia de la preprocesamiento de datos y cómo preparar conjuntos de datos para su uso en modelos de machine learning. Los estudiantes explorarán ejemplos prácticos utilizando datasets reales, lo que les permitirá aplicar teoría a práctica y entender mejor el flujo completo del proceso de modelado.

## Ejercicio
Implementa un modelo de regresión lineal usando scikit-learn en Python para predecir el precio de casas basado en características como el tamaño del terreno, número de habitaciones y ubicación. Utiliza validación cruzada para ajustar los hiperparámetros del modelo y evalúa su rendimiento utilizando métricas como el error cuadrático medio (MSE) y el coeficiente de determinación \(R^2\).

## Resumen
- Implementar modelos de regresión lineal con scikit-learn.
- Utilizar validación cruzada para ajustar hiperparámetros.
- Evaluar el rendimiento del modelo utilizando métricas apropiadas.
