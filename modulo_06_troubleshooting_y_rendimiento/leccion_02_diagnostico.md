# Diagnostico

## Objetivos
- Comprender los fundamentos del diagnóstico de sistemas.
- Identificar y resolver problemas comunes mediante técnicas básicas de diagnóstico.

## Contenido
El diagnóstico es una habilidad crucial en el mantenimiento y optimización del rendimiento de cualquier sistema. En este módulo, se abordarán los aspectos teóricos y prácticos necesarios para identificar problemas en un sistema informático o de red. Se explicará cómo utilizar herramientas de diagnóstico básicas, como comandos de línea y monitores de rendimiento, para analizar el comportamiento del sistema y localizar posibles fallos. Además, se discutirá la importancia de documentar los pasos de diagnóstico y las conclusiones obtenidas.

## Ejercicio
El alumno deberá diagnosticar un problema de rendimiento en una máquina virtual utilizando herramientas como `top` o `htop`. Deberá identificar el proceso que está consumiendo más recursos, analizar su impacto en la CPU y memoria, y documentar los pasos seguidos para llegar a esta conclusión.

## Resumen
- El diagnóstico es fundamental para resolver problemas de rendimiento.
- Se deben utilizar herramientas específicas para analizar el sistema.
- La documentación de los pasos de diagnóstico es crucial para la eficiencia del proceso.
