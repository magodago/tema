# Casos de uso reales

## Objetivos
- Analizar casos de uso reales que demuestren la aplicación del tema estudiado.
- Identificar las ventajas y desventajas de los casos de uso analizados.

## Contenido
En esta lección, profundizaremos en el análisis de casos de uso reales para ilustrar mejor el contenido teórico aprendido. Por ejemplo, consideremos un sistema de gestión de inventario en una cadena de supermercados. Este caso de uso demuestra la importancia de la optimización del stock y la reducción de costos operativos. Sin embargo, también se identifican desafíos como la precisión de los datos ecológicos y el impacto en la experiencia del cliente. Otro ejemplo es un sistema de recomendación personalizada para plataformas de streaming, que muestra cómo el algoritmo de filtrado colaborativo puede mejorar la satisfacción del usuario, pero también puede llevar a efectos de sesgo si no se implementa adecuadamente.

## Ejercicio
El alumno deberá seleccionar un caso de uso real relacionado con el tema estudiado y realizar una presentación breve (3-5 minutos) donde explique:
1. El contexto y la problemática.
2. La solución propuesta y su implementación.
3. Ventajas y desventajas del caso de uso.

## Resumen
- Se analizaron casos reales que ilustran el tema estudiado.
- Se identificaron tanto las ventajas como los desafíos asociados a estos casos.
